\textbf{R:} Dado que los atributos dados son variables continuas, nos apoyaremos de Bayes Naïve Gaussiano para determinar la clase a partir de las observaciones, por lo que usaremos la siguiente fórmula

Comenzaremos por obtener las probabilidades a priors de cada clase, entonces tenemos:

\begin{align*}
    P(Y=0)& = \frac{4}{9} = 0.444 \\
    P(Y=1) & = \frac{5}{9} = 0.556 \\
\end{align*}

% \mu es la media
Ahora procedemos a obtener las medias de cada clase para cada dato, partiendo de la fórmula $\frac{1}{N} \sum_{x} x$ entonces tenemos

\begin{align*}
    \mu_{y=0,x=1} & = \frac{0.5+0.3+0.2+0.4}{4} = 0.35 \\
    \mu_{y=0,x=2} & = \frac{0.6+0.6+0.5+0.6}{4} = 0.575 \\
    \mu_{y=0,x=3} & = \frac{0+0.01+0+0.1}{4} = 0.0275 \\\\
    \mu_{y=1,x=1} & = \frac{0.6+0.9+0.8+0.95+0.5}{5} = 0.75 \\
    \mu_{y=1,x=2} & = \frac{0.8+0.7+0.65+0.9+0.6}{5} = 0.73 \\
    \mu_{y=1,x=3} & = \frac{0.7+1+0.9+1+0.5}{5} = 0.82 \\
\end{align*}

% \sigma^2 es varianza de los datos
Ahora procedemos a obtener las varianzas de cada clase para cada dato, partiendo de la fórmula $\frac{1}{N-1} \sum_{x} (x-\mu_{y,x})^2$, entonces tenemos
\begin{align*}
    \sigma^2_{y=0,x=1} & = \frac{(0.5-0.35)^2 + (0.3-0.35)^2 + (0.2-0.35)^2 + (0.4-0.35)^2}{3} = 0.0167 \\
    \sigma^2_{y=0,x=2} & = \frac{(0.6-0.575)^2 + (0.6-0.575)^2 + (0.5-0.575)^2 + (0.6-0.575)^2}{3} = 0.0025 \\
    \sigma^2_{y=0,x=3} & = \frac{(0-0.0275)^2 + (0.01-0.0275)^2 + (0-0.0275)^2 + (0.1-0.0275)^2}{3} = 0.00235
\end{align*}
\begin{align*}
    \sigma^2_{y=1,x=1} & = \frac{(0.6-0.75)^2 + (0.9-0.75)^2 + (0.8-0.75)^2 + (0.95-0.75)^2 + (0.5-0.75)^2}{4} = 0.0375 \\
    \sigma^2_{y=1,x=2} & = \frac{(0.8-0.73)^2 + (0.7-0.73)^2 + (0.65-0.73)^2 + (0.9-0.73)^2 + (0.6-0.73)^2}{4} = 0.0145 \\
    \sigma^2_{y=1,x=3} & = \frac{(0.7-0.82)^2 + (1-0.82)^2 + (0.9-0.82)^2 + (1-0.82)^2 + (0.5-0.82)^2}{4} = 0.047 \\
\end{align*}

Ahora obtenremos el modelo de Bayes Naïve Gaussiano, para esto usaremos la fórmula \[P(x|y) = \frac{1}{\sqrt{2\pi\sigma^2_y}} e^{-\frac{(x-\mu_y)^2}{2\sigma^2_y}}\] 
Después aplicaremos la fórmula para la probabilidad conjunta
\[ P(y, x_1, x_2, ... , x_n) = P(y) \prod_{i=1}^{n} P(x_i | y) \]

Ahora evaluaremos las métricas, basandonos en la documentación de \href{https://scikit-learn.org/stable/api/sklearn.metrics.html}{\texttt{sklearn}} veamos que $tp$ es el número de verdaderos positivos (predijo $0$ y era $0$ o predijo $1$ y era $1$), $fp$ es el número de falsos positivos (predijo $0$ y era $1$) y $fn$ es el número de falsos negativos (predijo $1$ y era $0$). Por lo que tenemos
\begin{itemize}
    \item \textbf{Presicion:} Dada una clase, es el número de verdaderos positivos sobre la suma de verdaderos positivos y falsos positivos \\
    Para $Y=0$ 
    \[\frac{tp}{tp + fp} = \frac{3}{3 + 0} = 1\] 
    Para $Y=1$
    \[\frac{tp}{tp + fp} = \frac{2}{2 + 0} = 1\]

    \item \textbf{Exhaustividad (Recall):} Dada una clase, es el número de verdaderos positivos sobre la suma de verdaderos positivos y falsos negativos \\
    Para $Y=0$
    \[\frac{tp}{tp+fn} = \frac{3}{3+0} = 1\]
    Para $Y=1$
    \[\frac{tp}{tp+fn} = \frac{2}{2+0} = 1\]

    \item \textbf{F1-Score:} Dada una clase, es la \textit{harmonic mean} de la precisión y la exhaustividad \\
    Para $Y=0$
    \[2 \cdot \frac{\text{precisión} \cdot \text{recall}}{\text{precisión} + \text{recall}} = 2 \cdot \frac{1 \cdot 1}{1 + 1} = 1\]
    Para $Y=1$
    \[2 \cdot \frac{\text{precisión} \cdot \text{recall}}{\text{precisión} + \text{recall}} = 2 \cdot \frac{1 \cdot 1}{1 + 1} = 1\]

    \item \textbf{Accuracy:} Dadas todas las clases, es el número de verdaderos positivos sobre el total de ejemplos
    \[\frac{tp_{Y=0} + tp_{Y=1} }{N}= \frac{3 + 2}{5} = 1\]
    
    \item \textbf{Macro Average:} Es la media aritmética de cada uno de cada métrica para cada clase
    \begin{itemize}
        \item \textbf{Precision:} \[\frac{Precision_{Y=0} + Precision_{Y=1}}{2} = \frac{1 + 1}{2} = 1\]
        \item \textbf{Recall:} \[\frac{Recall_{Y=0} + Recall_{Y=1}}{2} = \frac{1 + 1}{2} = 1\]
        \item \textbf{F1-Score:} \[\frac{F1-Score_{Y=0} + F1-Score_{Y=1}}{2} = \frac{1 + 1}{2} = 1\]
    \end{itemize}
    
    \item \textbf{Weighted Average:} Se multiplica la métrica de cada clase por su peso proporcional (Soporte, número de ejemplares de cada clase) en el conjunto total.
        \begin{itemize}
            \item \textbf{Precision:} \[\frac{\sum_{i=0}^{1} \text{precisión}_i \cdot w_i }{\sum_{i=0}^{1} w_i} = \frac{(1 \cdot 3) + (1 \cdot 2)}{3 + 2} = \frac{5}{5} = 1\]
            
            \item \textbf{Recall:} \[\frac{\sum_{i=0}^{1} \text{recall}_i \cdot w_i }{\sum_{i=0}^{1} w_i} = \frac{(1 \cdot 3) + (1 \cdot 2)}{3 + 2} = \frac{5}{5} = 1\]
            
            \item \textbf{F1-Score:} \[\frac{\sum_{i=0}^{1} \text{F1-Score}_i \cdot w_i }{\sum_{i=0}^{1} w_i} = \frac{(1 \cdot 3) + (1 \cdot 2)}{3 + 2} = \frac{5}{5} = 1\]
        \end{itemize}
\end{itemize}
